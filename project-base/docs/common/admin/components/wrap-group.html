Wrapping box around fieldset

Special box for wrapping form groups with H2 heading or place with different color


.wrap-group {
	margin-bottom: 40px;
	padding: 20px;
	position: relative;

	background: @bg-box;
}

.wrap-group h2 {
	margin: 0 0 20px;

	font-size: 14px;
	font-weight: bold;
	text-transform: uppercase;
}

.wrap-group__inner {
	background: darken(@bg-box, 5%);
	padding: 10px;
}
